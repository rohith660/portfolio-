Cardiovascular Analysis Using Deep Learning

Angioplasty Project Overview This project aimed to enhance cardiovascular analysis by utilizing cutting-edge deep learning algorithms like ResNet18 and AlexNet to improve the accuracy and precision of rotational atherectomy angioplasty procedures. Angioplasty, being a minimally invasive technique, requires detailed imaging analysis to ensure successful treatment of arterial blockages. By integrating TensorFlow and Keras, I trained models to predict and classify medical imaging data, helping physicians make more informed decisions. The project focused on optimizing patient outcomes through the advanced predictive capabilities of deep learning and high-quality data analysis. Tools & Technologies: ResNet18 and AlexNet (Deep Learning Algorithms) TensorFlow and Keras (Model training and evaluation) Python (Programming language) Medical Imaging Datasets (For training and testing models)
